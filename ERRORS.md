## Ошибки

### 1. Результат дан только в чат, файл не создан
- **Проблема:** Аудит Instagram и отчёт scout_market (13.03.2026) даны только в чат — при закрытии сессии исчезли
- **Причина:** Не было правила "файл создаётся ДО начала работы". Работал без ограничения.
- **Решение:** Правило закреплено в CLAUDE.md (правило сохранения), MEMORY.md (долгосрочная память). Файл создаётся в момент открытия задачи, не в конце.

### 2. Протокол сохранён там где не читается
- **Проблема:** Файл `skills/session_protocol.md` создан как "протокол", но Claude его не читает автоматически
- **Причина:** Не проверил что файл попадёт в автозагрузку
- **Решение:** Все критичные правила — только в CLAUDE.md и MEMORY.md. skills/ — для инструкций скиллов, не для правил Claude.

### 3. Аудит по памяти без чтения источников
- **Проблема:** Анализ или рекомендации даются без чтения всех файлов по теме
- **Причина:** Нет правила "сначала прочитай всё, потом делай выводы"
- **Решение:** Добавлено в CLAUDE.md (блок "Качество анализа"). Использовать refs/AUDIT_CHECKLIST.md.
